## Getting started

`cd` into your new folder and run:
- `npm install`

To start the app run:

- `npm run build-prod`
- `npm run start`

For Development mode run:

- `npm run build-dev`

For Tests:

- `npm run test`

Extra Functionality:

- `pictures for rural locations`
